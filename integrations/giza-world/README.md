# Giza World integration bridge

Copy `giza-world-bridge.js` into `Asnow365/giza-world` and wire its adapter hooks to the real scene switcher, character selection, animation mixer, T2M prompt box, HUD, and accuracy panel.

The terminal environment could not clone the GitHub repo because the CONNECT tunnel returned `403`, so this bridge is designed as a copy-ready integration layer rather than an applied patch.
